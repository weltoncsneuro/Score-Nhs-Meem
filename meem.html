Mini Exame do Estado Mental (MEEM)

1. Orientação temporal (0 a 5 pontos)

Faça perguntas relacionadas ao tempo. Pontue 1 para cada resposta correta na primeira tentativa.

2. Orientação espacial (0 a 5 pontos)

Pergunte sobre o local onde o paciente está. Pontue 1 para cada resposta correta.

3. Memória imediata (0 a 3 pontos)

Peça ao paciente para repetir as palavras "Vaso", "Carro" e "Tijolo". Pontue 1 para cada palavra correta na primeira tentativa.

4. Atenção e cálculo (0 a 5 pontos)

Peça ao paciente para realizar subtrações seriadas de 7, começando em 100. Pontue 1 para cada resposta correta.

5. Evocação (0 a 3 pontos)

Peça ao paciente para recordar as palavras "Vaso", "Carro" e "Tijolo". Pontue 1 para cada palavra correta.

6. Linguagem (0 a 6 pontos)

Inclui nomeação, repetição e comandos. Pontue 1 para cada resposta correta.

7. Escrita (0 a 1 ponto)

Peça ao paciente para escrever uma frase completa com sentido. Pontue 1 se o objetivo for atingido.

8. Visuoespacial (0 a 1 ponto)

Peça ao paciente para copiar dois pentágonos que se cruzam. Pontue 1 se a figura estiver correta.
